# gl-list

### Requirements
- [CocoaPods](https://cocoapods.org/)

### Clone this repo
Clone this repository using:
```sh
$ git clone git@bitbucket.org:nicowez/gl-list.git
```
or
```sh
$ git clone https://nicowez@bitbucket.org/nicowez/gl-list.git
```
### Open project
```sh
$ cd GlobalLogicList/
$ pod install
$ open GlobalLogicList.xcworkspace
```
run!
